# BIP-EDNEI

Projeto full-stack para estudo e demonstração de integração entre:

- Backend REST (Spring Boot)
- Módulo EJB (WildFly)
- Frontend Angular
- Integração entre camadas via HTTP

## 📌 Objetivo

Criar solução completa em camadas (DB, EJB, Backend, Frontend), corrigindo bug em EJB e entregando aplicação funcional.

## 🏗 Estrutura do Projeto
bip-ednei/
│
├── bip-ednei-backend → API REST (Spring Boot)
├── bip-ednei-ejb → Serviços EJB (WildFly)
├── bip-ednei-frontend → Aplicação Angular
└── README.md

## 🔧 Tecnologias Utilizadas

### Backend
- Java 21
- Spring Boot
- Maven
- Hibernate / JPA

### EJB Module
- Jakarta EE
- WildFly
- EJB Stateless

### Frontend
- Angular
- TypeScript
- Node.js
